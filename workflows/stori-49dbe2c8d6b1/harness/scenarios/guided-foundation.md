# Guided Foundation Setup

## Why This Scenario Exists

Stori's first wedge is helping a creator start correctly from a blank slate. If this flow is weak, the product loses its activation advantage.

## Representative User

A solo indie game developer opens Stori with no existing world data and wants a structured starting point without feeling trapped by a rigid wizard.

## Task

Guide the user from an empty workspace through a skippable setup flow that captures the world's premise, tone, foundational rules, and a small starter set of entities.

## Expected Outcome

- the user can skip any step without getting stuck
- the wizard produces a usable initial world structure instead of an empty dashboard
- the resulting workspace reflects the chosen premise and starter entities
- the user can immediately continue editing after setup

## Validation Ideas

- start from a new world with no prior data
- complete the wizard with realistic answers
- repeat while skipping at least half the steps
- verify that a starter world still exists in both paths

## Failure Signals

- setup feels like form-filling instead of creative scaffolding
- the user exits the wizard into an empty or confusing workspace
- skipped paths produce broken defaults or missing required entities
